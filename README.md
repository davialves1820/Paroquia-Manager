# Paróquia Manager

Bem-vindo ao repositório do **Paróquia Manager**, um sistema completo para gestão administrativa e pastoral da sua paróquia.

## 📂 Estrutura do Projeto

O projeto é dividido em dois repositórios principais:

1.  **[Backend](./backend)**: API robusta feita em AdonisJS 6 com PostgreSQL.
    - [Ver README de Usuário](./backend/README.md)
    - [Ver README de Desenvolvedor](./backend/README.dev.md)

2.  **[Frontend](./frontend/frontend)**: Interface moderna e rápida feita em Next.js.
    - [Ver README de Usuário](./frontend/frontend/README.md)
    - [Ver README de Desenvolvedor](./frontend/frontend/README.dev.md)

## 🚀 Como começar rápido

1.  Configure e inicie o **Backend** seguindo o [Guia do Desenvolvedor Backend](./backend/README.dev.md).
2.  Configure e inicie o **Frontend** seguindo o [Guia do Desenvolvedor Frontend](./frontend/frontend/README.dev.md).
3.  Acesse `http://localhost:3000` e comece a gerenciar!
