# Paróquia Manager - Backend

Sistema de gerenciamento paroquial, desenvolvido com **AdonisJS 6**, focado em automação de processos, relatórios e controle de sacramentos.

## ✨ Funcionalidades Principais

- **Gestão de Catequese:** Controle de turmas, alunos, frequências e status sacramental.
- **Relatórios Automatizados:** Geração de PDFs e planilhas Excel para controle administrativo.
- **Controle de Sacramentos:** Monitoramento completo de Batismos e Primeiras Eucaristias.
- **Documentação Automática:** API documentada via Swagger para fácil integração.
- **Segurança:** Sistema de autenticação e permissões granuladas.

## 🚀 Como começar (Usuário)

Para utilizar o sistema, certifique-se de que o backend está rodando e acesse a interface frontend. A documentação das rotas disponíveis pode ser acessada via:
`http://localhost:3333/docs` (em ambiente de desenvolvimento).
